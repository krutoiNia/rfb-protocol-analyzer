# Асинхронный анализатор безопасности протокола RFB/VNC (v4.1)

Высокопроизводительный асинхронный сетевой сканер, разработанный для системных администраторов и специалистов по информационной безопасности. Инструмент предназначен для аудита надежности паролей и проверки конфигураций доступа в различных расширениях протокола RFB/VNC.

**Дисклеймер:** Данное программное обеспечение создано исключительно в образовательных целях и для проведения авторизованного тестирования на проникновение (пентеста) внутри сетей, принадлежащих вам, либо при наличии официального разрешения. Автор не несет ответственности за любой ущерб или неправомерное использование данного инструмента.

## Ключевые инженерные особенности
* **Асинхронная архитектура:** Движок на базе `asyncio` и `uvloop` обеспечивает сверхбыструю трехэтапную обработку пула хостов (`Прескан` → `Зондирование` → `Оценка`).
* **Глубокое покрытие протоколов:** Реализация низкоуровневого обмена криптографическими ключами для стандартного VNC, Apple ARD (Diffie-Hellman и RC4), TightVNC, UltraVNC MSLogon I/II и VeNCrypt (с TLS-туннелированием).
* **Адаптивный Rate Limiting:** Интеллектуальный контроль общей пропускной способности через алгоритм `TokenBucket` и механизм экспоненциальной задержки `HostGate` для корректного обхода скрытых блокировок сессий (`silent-close`).
* **Информативный дашборд:** Интерактивный консольный интерфейс на базе библиотеки `rich` с динамическим расчетом ETA и графиками скорости (RPS) в реальном времени.

## Установка зависимостей
```bash
pip install rich cryptography uvloop
```

## Легальное использование (Пример команды)
